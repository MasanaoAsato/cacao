# review level
You can specify the level of review you want:
- critical: Focuses on critical issues that could cause bugs or security vulnerabilities. Must be fixed.
- major: Focuses on design issues, performance problems, or deviations from specifications. Recommended to fix.
- minor: Focuses on readability, naming, or slight style violations. Optional to fix.
- suggestion: Focuses on better alternatives or learning-oriented feedback. For reference only.
